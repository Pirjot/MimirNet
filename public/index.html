Welcome to MimirNet!

MimirNet is a simple multilayered neural network, developed with a "layer-focused" implementation. This description is quite short, so I'll just link the main source from where I learned how to develop this: The Coding Train!

I made this project mostly to serve as my own intro to Machine Learning and AI, but it also has the plus of being a library that can be used for neural networks as well! You can use this neural network to a lot of things, and a few examples are provided below. Have fun!
